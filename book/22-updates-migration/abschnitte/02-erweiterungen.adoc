diese nun auf 3.x migrieren? Wenn Sie sich an das MVC-Framework gehalten
haben, reicht es, die in der Dokumentation beschriebenen Schritte
durchzugehen, um die eigene Erweiterung an die API-Änderungen
anzupassen:
